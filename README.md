# 基于springboot的就业信息管理系统5



## 1、项目介绍

基于springboot的就业信息管理系统5拥有四种角色：管理员、学生、导师和企业

![结构](https://www.codeshop.fun/Typora-Images/202402171858589.jpg)

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：html、VUE

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
